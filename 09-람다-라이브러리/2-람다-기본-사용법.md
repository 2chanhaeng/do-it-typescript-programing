## `tap`

첫번째 인자로 넘어온 콜백함수에 두번째 인자로 들어온 값을 넘겨 실행만 한 뒤, 두번째 인자로 들어온 값을 그대로 반환한다.  
`console.log` 를 이용해 디버깅을 할 때 유용하게 사용할 수 있다.

```ts
import { tap } from "ramda";
const one = tap((n) => console.log(`logged by tap ${n}`), 1); // logged by tap 1
console.log(one); // 1
```
